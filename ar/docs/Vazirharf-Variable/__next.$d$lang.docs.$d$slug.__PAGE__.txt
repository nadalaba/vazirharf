3:T11f8,<h2>ما هو الخط المتغير؟</h2>
<p>الخط المتغير هو نسخة متقدمة من خطوط (OpenType) التقليدية، حيث يمكنه احتواء أنماط متعددة من الخطوط داخل ملف واحد فقط. هذا يتيح للمستخدم استخدام ملف واحد بدلاً من ملفات متعددة لكل نمط. على سبيل المثال، يمكن أن يتضمن الخط المتغير الأوزان العادية والعريضة، والنمط المائل، والمضغوط، والممتد، وغيرها، كلها في ملف واحد. بينما في السابق كنا نحتاج إلى استخدام عدة ملفات للوصول إلى هذه الأنماط.</p>
<h2>ملفات الخط المتغير لوزیرحرف</h2>
<p>تتوفر نسخ مختلفة من خط وزیرحرف المتغير في المسارات التالية:</p>
<h3>النسخة العادية:</h3>
<ul>
<li><code>fonts/variable/Vazirharf[wght].ttf</code></li>
<li><code>fonts/webfonts/Vazirharf[wght].woff2</code></li>
</ul>
<h3>نسخة النقاط الدائرية:</h3>
<ul>
<li><code>Round-Dots/fonts/variable/Vazirharf-RD[wght].ttf</code></li>
<li><code>Round-Dots/fonts/webfonts/Vazirharf-RD[wght].woff2</code></li>
</ul>
<h3>النسخة بدون حروف لاتينية:</h3>
<ul>
<li><code>misc/Non-Latin/fonts/variable/Vazirharf-NL[wght].ttf</code></li>
<li><code>misc/Non-Latin/fonts/webfonts/Vazirharf-NL[wght].woff2</code></li>
</ul>
<p>كل من هذه الملفات يحتوي على جميع أوزان خط وزیرحرف في حزمة واحدة.</p>
<h2>كيفية استخدام خط وزیرحرف المتغير على الويب</h2>
<p>لتعريف الخط في CSS:</p>
<pre><code class="language-css">@font-face {
  font-family: 'Vazirharf';
  src: url('fonts/webfonts/Vazirharf[wght].woff2') format('woff2 supports variations'),
       url('fonts/webfonts/Vazirharf[wght].woff2') format('woff2-variations');
  font-weight: 100 900;
  font-style: normal;
  font-display: swap;
}
</code></pre>
<p>لنسخة النقاط الدائرية:</p>
<pre><code class="language-css">@font-face {
  font-family: 'Vazirharf RD';
  src: url('Round-Dots/fonts/webfonts/Vazirharf-RD[wght].woff2') format('woff2 supports variations'),
       url('Round-Dots/fonts/webfonts/Vazirharf-RD[wght].woff2') format('woff2-variations');
  font-weight: 100 900;
  font-style: normal;
  font-display: swap;
}
</code></pre>
<p>يمكن أيضًا استخدام CDN بدلاً من المسارات المحلية، مثل:</p>
<pre><code class="language-css">@font-face {
  font-family: 'Vazirharf RD';
  src: url('https://cdn.jsdelivr.net/gh/nadalaba/vazirharf@v34.002/Round-Dots/fonts/webfonts/Vazirharf-RD[wght].woff2') format('woff2 supports variations'),
       url('https://cdn.jsdelivr.net/gh/nadalaba/vazirharf@v34.002/Round-Dots/fonts/webfonts/Vazirharf-RD[wght].woff2') format('woff2-variations');
  font-weight: 100 900;
  font-style: normal;
  font-display: swap;
}
</code></pre>
<p>بالإضافة إلى ذلك، يوجد في المجلد الأساسي ملف باسم <code>Vazirharf-Variable-font-face.css</code> مخصص للاستخدام على الويب، ويمكن تضمينه مباشرة أو عبر CDN:</p>
<pre><code class="language-html">&#x3C;link href="https://cdn.jsdelivr.net/gh/nadalaba/vazirharf@v34.002/Vazirharf-Variable-font-face.css" rel="stylesheet" type="text/css" />
</code></pre>
<p>ملاحظة: يُفضل دائمًا استخدام أحدث إصدار بدلاً من رقم الإصدار الثابت v34.002.</p>
<p>الاستخدام في CSS يكون كالتالي:</p>
<pre><code class="language-css">body {
    font-family: Vazirharf;
    font-weight: 400;
}

h1 {
    font-weight: 900;
}
</code></pre>
<p>يمكن استخدام أي رقم بين 100 و900 لتغيير الوزن، مثل 692.</p>
<p>أو استخدام معامل <code>wght</code>:</p>
<pre><code class="language-css">body {

    font-variation-settings: "wght" 400;

h1 {
    font-variation-settings: "wght" 900;
}
</code></pre>
<p>في خط وزیرحرف، تم تعريف معامل <code>wght</code> للتحكم في وزن الخط. يمكن استخدام أي قيمة بين 100 و900، مثل 627.</p>
